# 🧪 دليل الاختبار السريع - النظام الذكي

## 🎯 اختبار النظام الأساسي

### 1. اختبار التحميل التلقائي:
```
✅ افتح التطبيق لأول مرة
✅ ضع رابط ملف Excel: https://yoursite.com/products.xlsx
✅ اضغط "إعداد مصدر البيانات"
✅ تأكد من ظهور رسالة "تم تحميل X منتج بنجاح"
✅ أغلق التطبيق وافتحه مرة أخرى
✅ يجب أن تظهر رسالة "تم تحميل X منتج من الذاكرة المحلية"
```

### 2. اختبار العمل بدون انترنت:
```
✅ تأكد من تحميل البيانات (خطوة 1)
✅ افصل الانترنت من الجهاز
✅ أعد فتح التطبيق
✅ يجب أن يعمل التطبيق بشكل طبيعي
✅ جرب مسح باركود
✅ يجب أن يجد المنتجات بدون مشاكل
```

### 3. اختبار إشعارات التحديث:
```
✅ تأكد من تحميل البيانات
✅ قم بتحديث ملف Excel على الخادم
✅ انتظر 5 دقائق أو أعد تحميل الصفحة
✅ يجب أن تظهر رسالة "تحديث متاح"
✅ اضغط "تحديث الآن"
✅ تأكد من تحديث البيانات
```

## 📱 اختبار وظائف المسح

### 4. اختبار مسح الباركود:
```
✅ اضغط "تشغيل الكاميرا"
✅ امسح باركود معروف (موجود في ملف Excel)
✅ يجب أن تظهر شاشة "قل الكمية المطلوبة"
✅ امسح باركود غير معروف
✅ يجب أن تظهر رسالة "كود غير معروف"
```

### 5. اختبار التعرف على الصوت:
```
✅ بعد مسح باركود صحيح، اضغط "ابدأ التسجيل"
✅ قل "خمسة" بوضوح
✅ يجب أن تظهر شاشة التأكيد مع الكمية 5
✅ جرب أرقام مختلفة: "واحد"، "عشرة"، "عشرين"
✅ جرب قول أرقام إنجليزية: "five"
✅ يجب أن يفهم الأرقام العربية فقط
```

## 🔧 اختبار المميزات المتقدمة

### 6. اختبار العد التنازلي:
```
✅ بعد إدخال الكمية بالصوت
✅ راقب العد التنازلي من 5 إلى 1
✅ يجب أن يضاف المنتج تلقائياً بعد انتهاء العد
✅ جرب الضغط على "تأكيد الإضافة" قبل انتهاء العد
✅ جرب الضغط على "إلغاء"
```

### 7. اختبار قائمة المنتجات:
```
✅ أضف عدة منتجات
✅ تأكد من ظهورها في قائمة "المنتجات المضافة"
✅ جرب حذف منتج بالضغط على "🗑️ حذف"
✅ تأكد من تحديث العدد في الإحصائيات
```

## 📊 اختبار البيانات والإحصائيات

### 8. اختبار الإحصائيات:
```
✅ تحقق من عرض "إجمالي المنتجات" في قاعدة البيانات
✅ تحقق من عرض "منتجات ممسوحة"
✅ أضف منتج وتأكد من تحديث الإحصائيات
✅ احذف منتج وتأكد من تحديث الإحصائيات
```

### 9. اختبار حفظ البيانات:
```
✅ أضف عدة منتجات
✅ أغلق التطبيق
✅ افتح التطبيق مرة أخرى
✅ يجب أن تظهر نفس المنتجات المضافة
✅ تأكد من عدم فقدان أي بيانات
```

## 🌐 اختبار الاتصال والمزامنة

### 10. اختبار استعادة الاتصال:
```
✅ ابدأ بدون انترنت
✅ جرب استخدام التطبيق
✅ قم بتوصيل الانترنت
✅ يجب أن تختفي رسالة "لا يوجد اتصال"
✅ يجب أن يبدأ التحقق من التحديثات تلقائياً
```

### 11. اختبار خصائص PWA:
```
✅ افتح التطبيق في Chrome على الهاتف
✅ يجب أن تظهر رسالة "إضافة إلى الشاشة الرئيسية"
✅ أضف التطبيق للشاشة الرئيسية
✅ افتح التطبيق من الشاشة الرئيسية
✅ يجب أن يعمل كتطبيق مستقل
```

## 🚨 اختبار معالجة الأخطاء

### 12. اختبار الأخطاء الشائعة:
```
✅ أدخل رابط خاطئ لملف Excel
✅ يجب أن تظهر رسالة خطأ واضحة
✅ جرب ملف Excel بتنسيق خاطئ
✅ يجب أن يتعامل مع الخطأ بلطف
✅ جرب منع إذن الكاميرا
✅ يجب أن تظهر رسالة مفيدة
✅ جرب منع إذن الميكروفون
✅ يجب أن تظهر رسالة مفيدة
```
